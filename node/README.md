# @olvid/bot-node #

## Introduction
This repository is part of the Olvid bots framework.
If you're new here, consider starting with our [Documentation](https://doc.bot.olvid.io).

This module allows you to connect with an Olvid daemon and create bots using typescript or node.js.

## Installation

```bash
npm install @olvid/bot-node
```

## Usage
Follow our [🚀 Quickstart](https://doc.bot.olvid.io/install) page to set up your Olvid daemon instance. Then you can start coding following our [Js Client](https://doc.bot.olvid.io/js) documentation page.
